# 오늘뽁 · 문구(카피) 정리

조건별로 달라지는 모든 문구 모음. **수정 시 실제 소스도 함께 고칠 것.**
소스 위치: 인사말 = `bubble-pop-lab/src/main.ts` (`GREETINGS_F`) · 칭찬/결과/레벨/마일스톤 = `bubble-pop-lab/src/modes.ts` · 결과 제목 = `bubble-pop-lab/src/main.ts` · 온보딩 = `bubble-pop-lab/index.html`.

## 1. 홈 인사말 — 시간대별 (7종)
| 시간대 | 문구 |
|---|---|
| 새벽 (0~6시) | 아직 안 잔거야? 좀 쉬어 |
| 이른 아침 (6~9시) | 좋은 아침! 한 판 하고 가 |
| 오전 (9~12시) | 잠깐 쉬어가, 한 판 어때? |
| 점심 (12~14시) | 점심 먹었어? 밥 잘 챙겨먹어! |
| 오후 (14~18시) | 거의 다 왔어, 여기 두고 가 |
| 저녁 (18~21시) | 고생했어, 스트레스 털고 가자 |
| 밤 (21~24시) | 수고했어, 다 터뜨리고 자 |

## 2. 결과 화면 제목 — 모드/레벨별
- **스트레스 터뜨리기**: `다 뿌셨어요!`
- **몰입 모드**: Lv.10↑ → `몰입 마스터 달성!` / 그 외 → `(걸린시간) 동안 몰입!`
- **명상 모드**: Lv.10↑ → `뽁뽁 마스터 달성!` / 그 외 → `시원하게 비웠어요`

## 3. 결과 칭찬 카피 — 명상·몰입 (랜덤 12종)
1. 뽁뽁 터진 만큼, 스트레스도 터져 나갔어요.
2. 손가락이 움직인 만큼, 마음이 가벼워졌어요.
3. 오늘의 스트레스, 다 비워냈어요.
4. 뽁뽁 소리에 걱정이 사라졌어요.
5. 한 개 한 개 스트레스가 톡톡 빠져나갔어요.
6. 머릿속이 한결 맑아졌을 거예요.
7. 손끝에서 시작된 작은 해방.
8. 뽁! 하나 터질 때마다 하나씩 내려놓은 거예요.
9. 오늘도 잘 비웠어요. 내일 또 만나요.
10. 마음이 조금 가벼워졌길 바라요.
11. 스트레스 제로. 지금 이 순간, 충분해요.
12. 잘했어요. 오늘 하루, 수고했어요.

- **Lv.10 올클리어 (고정)**:
  - 몰입: `(시간) 만에 올클리어!\n완벽한 집중력이었어요.`
  - 명상: `10단계 전부 클리어!\n오늘 스트레스, 완전히 날려버렸어요.`

## 4. 스트레스 터뜨리기 칭찬 — 입력 이름(OOO) 포함 (랜덤 5종)
1. "OOO" 다 부쉈어요!\n이제 없어요, 시원하죠?
2. OOO?\n뽁뽁 터뜨려서 날려버렸어요.
3. OOO 따위,\n손가락으로 전부 없앴어요.
4. OOO는 이제 가루가 됐어요.\n오늘 하루 수고했어요.
5. 뽁! 뽁! 뽁!\nOOO, 안녕~

## 8. 온보딩 — 3장 (최초 실행 시)
1. **오늘의 스트레스, 뽁뽁 터뜨려서 날려버리세요** / 손끝으로 톡톡, 마음속 답답함도 함께 터뜨려보세요.
2. **상황에 맞게 3가지 모드를 즐겨보세요** / 차분하게 명상하거나, 집중해서 몰입하거나, 스트레스를 시원하게 터뜨려보세요.
3. **오늘의 스트레스, 다 터뜨렸나요?** / 오늘 기록, 이번 달 기록, 레벨업까지 한눈에 확인할 수 있어요.
