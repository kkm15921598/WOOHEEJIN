# 🅳 버블팝랩 (BubblePopLab)

> 무한 뽁뽁이 + 일일 한정 패턴 + 1분 챌린지

## 컨셉
화면 가득 펼쳐진 3D 버블을 손가락으로 *뽀삭* 터트리며 스트레스를 해소하는 새티스파잉 캐주얼 게임.

## 모드
- **명상 모드**: 무한, 타이머 없음
- **1분 챌린지**: 베스트 기록 갱신

## 기술 스택
- Vite + TypeScript
- Three.js (vanilla)
- Web Audio API + navigator.vibrate
- vite-plugin-pwa

## 배포
- 웹: Vercel
- 앱: PWABuilder.com → APK → Google Play
